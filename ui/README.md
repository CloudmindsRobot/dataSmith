### 目录结构

```text
┌──src                
│   ├──api            * api接口
│   ├──assets         * 静态资源
│   ├──cross-ui       * ui组件库
│   ├──locale         * 多语言
│   ├──mixins         
│   ├──routes         * 路由
│   └──views          * 页面
├──.env.development        
├──.env.production       
├──babel.config.js           
├──package.json       
└──vue.config.js 
```

### 安装依赖

```
npm install
```

### 运行

```
npm run serve
```

### 编译生成

```
npm run build
```
